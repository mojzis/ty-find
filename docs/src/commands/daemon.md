# daemon

Manage the background LSP server (auto-starts on first use)

## Usage

```
tyf daemon
```

## Subcommands

**`start`**
: Start the background LSP server

**`stop`**
: Stop the background LSP server

**`status`**
: Show the daemon's running status

## Examples

```bash
# Start the background daemon
tyf daemon start

# Check daemon status
tyf daemon status

# Stop the daemon
tyf daemon stop
```
