# document-symbols

List all symbols in a file

## Usage

```
ty-find document-symbols <FILE>
```

## Arguments

| Argument | Description |
|----------|-------------|
| `<file>` |  *(required)* |

## Examples

```bash
# List all symbols in a file
ty-find document-symbols main.py
```
